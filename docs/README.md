\# Документация проекта Калькулятор



\## Описание

Простой калькулятор с базовыми математическими операциями.



\## Функции



\### add(a, b)

Складывает два числа.



\*\*Параметры:\*\*

\- a (int/float): Первое число

\- b (int/float): Второе число



\*\*Возвращает:\*\* Сумму a и b



\*\*Пример:\*\*

```python

result = add(5, 3)  # 8

```



\### subtract(a, b)

Вычитает второе число из первого.



\### multiply(a, b)

Умножает два числа.



\### divide(a, b)

Делит первое число на второе.



\*\*Исключения:\*\*

\- ValueError: если b равно 0
